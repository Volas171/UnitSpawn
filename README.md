# Unit Spawn
![Build](https://github.com/Volas171/UnitSpawn/workflows/Build/badge.svg)
## Use

Template: `/spawn unit count team`

Ex: `/spawn mono 10 sharded`

## Installation
./gradlew jar 

## Precompiled binaries 

Check actions tab for .jar or release tab for src **(lmao)**
